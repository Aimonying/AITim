## 环境要求

### 必需软件
| 软件 | 版本要求 | 说明 |
|------|---------|------|
| Android Studio | 2022.0+ | Android 官方IDE |
| JDK | 17+ | Java 开发工具包 |
| Android SDK | API 33+ | Android 开发组件 |
| Gradle | 8.0+ | 项目构建工具 |

### 硬件要求
- **手机/模拟器**: Android 7.0 (API 24) 及以上
- **麦克风权限**: 应用需要麦克风进行录音
- **存储空间**: 建议预留 500MB+ 用于录音文件存储

## 项目导入

### 步骤 1: 打开 Android Studio
1. 启动 Android Studio
2. 点击左上角菜单栏 **File** → **Open**

### 步骤 2: 选择项目文件夹
1. 在弹出的文件选择器中
2. 找到并选择 `F:\APP\zhitingji\TingJiZhuShou` 文件夹
3. 点击 **OK**

```
📁 选择文件夹路径:
F:\APP\zhitingji\TingJiZhuShou\
├── app/
├── build.gradle          ← 选择 TingJiZhuShou 文件夹，不是这个文件！
├── build.gradle.kts
├── gradle/
├── gradle.properties
└── settings.gradle.kts
```

> ⚠️ **重要**: 选择的是 **文件夹**（TingJiZhuShou），不是里面的 build.gradle 文件！

### 步骤 3: 等待项目加载
1. 第一次打开会显示 "Building... 0%" 进度
2. 右下角会显示 "Gradle Sync" 进度条
3. 首次同步需要 5-15 分钟（取决于网络速度）

### 步骤 4: Gradle 同步操作

#### 如果同步未自动开始，手动触发：
1. 点击 Android Studio 工具栏的 🔄 **Sync Project with Gradle Files** 按钮
   - 位置：工具栏最右侧（大象图标旁边）
   - 快捷键：`Ctrl + Shift + O`（macOS）或 `Ctrl + O`（Windows）

#### 同步状态确认：
```
┌─────────────────────────────────────┐
│           Android Studio            │
├─────────────────────────────────────┤
│  🔄 Sync Project with Gradle Files │  ← 点击这个按钮
│                                     │
│  ✅ build.gradle (app)           OK │
│  ✅ build.gradle (Project)        OK │
│  ✅ settings.gradle.kts          OK │
└─────────────────────────────────────┘
```

#### 查看同步进度：
- 右下角状态栏显示 "Gradle sync finished"
- 或者显示 ✅ 勾号表示同步成功
- 如果显示 ❌ 红色错误，查看下方 [同步失败处理](#同步失败处理)

### 步骤 5: 检查 SDK 配置
1. 点击 **File** → **Project Structure**（或按 `Ctrl + Alt + Shift + S`）
2. 在左侧菜单选择 **Modules** → **app**
3. 确认以下配置：

| 配置项 | 值 |
|--------|-----|
| Compile SDK | 34 |
| Min SDK | 24 |
| Target SDK | 34 |
| JDK | 17 |

## 同步失败处理

### 错误: "Gradle sync failed: Connection refused"
**原因**: 网络问题，无法下载 Gradle 依赖

**解决方法**:
1. 检查网络连接
2. 重试点击 🔄 同步按钮
3. 或配置国内镜像源

### 错误: "SDK location not found"
**原因**: Android SDK 路径未配置

**解决方法**:
1. 点击 **File** → **Project Structure**
2. 选择 **SDK Location**
3. 设置 **Android SDK location** 为:
   ```
   C:\Users\你的用户名\AppData\Local\Android\Sdk
   ```

### 错误: "Unsupported class file version"
**原因**: JDK 版本过低

**解决方法**:
1. 点击 **File** → **Project Structure**
2. 选择 **SDK Location**
3. 设置 **JDK location** 为 JDK 17 安装路径

## 运行应用

### 方式一: 真机调试

#### 1. 手机设置
```
手机端操作:
设置 → 关于手机 → 连续点击"版本号"7次 → 开启开发者模式
设置 → 开发者选项 → 开启"USB调试"
```

#### 2. 连接电脑
- 使用 USB 数据线连接手机和电脑
- 手机弹出 "允许USB调试？" → 点击 **允许**

#### 3. 选择设备并运行
1. Android Studio 顶部工具栏
2. 点击设备选择器（手机图标旁边）

```
┌─────────────────────────────────────────────┐
│ ▶️ Run    📱 device |    🔧 TingJiZhuShou  │
│           ▼ 华为 P50 (R22)                  │  ← 选择你的手机
│             Pixel 6 API 34 (模拟器)         │
└─────────────────────────────────────────────┘
```

4. 选择你的手机设备
5. 点击 ▶️ **Run** 按钮（或按 `Shift + F10`）

### 方式二: 模拟器调试

#### 1. 打开设备管理器
- 菜单: **Tools** → **Device Manager**
- 或点击工具栏 📱 图标

#### 2. 创建模拟器
```
Device Manager 界面:
┌──────────────────────────────┐
│ + Create Device              │  ← 点击
└──────────────────────────────┘

步骤1: Select Hardware
- 选择 "Phone" → "Pixel 6" → Next

步骤2: Select System Image  
- 选择 "API 34" → "x86_64" → Download（首次需下载）
- 下载完成后点击 Next

步骤3: Verify Configuration
- 检查配置 → Finish
```

#### 3. 启动模拟器
```
在 Device Manager 中:
┌──────────────────────────────┐
│ ▶️ Pixel 6 API 34           │  ← 点击播放按钮启动
│   x86_64, API 34             │
└──────────────────────────────┘
```

#### 4. 运行应用到模拟器
1. 等待模拟器启动完成
2. 在 Android Studio 设备选择器中选择 "Pixel 6 API 34"
3. 点击 ▶️ **Run** 按钮

### 录音功能

#### 开始录音
1. 在首页点击 **开始录音** 按钮
2. 选择录音模式：
   - **实时录音 (REALTIME)**: 边录边转文字
   - **上传录音 (UPLOAD)**: 先录音后上传转写
   - **文本输入 (TEXT)**: 直接输入文字

#### 录音中
- 界面显示：
  - 🎤 录音波形动画
  - ⏱️ 已录时长
  - 📊 文件大小（上限 500MB）

#### 暂停/继续
- 点击 ⏸️ 暂停录音
- 再次点击 ▶️ 继续录音

#### 停止录音
- 点击 ⏹️ 停止录音
- 录音自动保存并开始转写

### 转写功能

#### 等待转写
- 显示转写进度条
- 完成后自动跳转到结果页面

#### 查看转写结果
- 显示识别的文字内容
- 支持复制和分享

### 历史记录

#### 浏览记录
- 首页底部点击 **历史** 标签
- 显示所有录音记录列表

#### 搜索记录
- 在搜索框输入关键词
- 支持按标题和内容搜索

#### 操作记录
- **查看**: 点击记录查看详情
- **收藏**: 点击 ⭐ 收藏重要记录
- **删除**: 长按记录，选择删除

### 设置

#### 语言设置
- **普通话**: zh-CN（默认）
- **英语**: en-US

#### 文件管理
- 查看存储空间使用情况

## 常见问题

### Q1: 录音时提示"无法访问麦克风"
**解决方法**：
1. 进入手机 **设置** → **应用**
2. 找到 **听记助手**
3. 点击 **权限**
4. 确保 **麦克风** 权限已开启

### Q2: 转写失败怎么办
**可能原因**：
- 网络连接不稳定
- 手机系统语音识别服务不可用

**解决方法**：
1. 检查网络连接
2. 重启应用重试
3. 确认手机系统语音服务可用

### Q3: 文件大小超限
**说明**：单次录音限制 500MB，超限后会自动停止录音并保存。

### Q4: 录音文件存储在哪里
**位置**：应用私有目录 `data/data/com.tingjizhushou/files/recordings/`

### Q5: 如何卸载应用
**方法**：
1. 在手机上长按应用图标
2. 拖动到卸载区域
3. 确认卸载

# 听记助手 APP 产品需求文档(PRD)

## 1. 产品概述

### 1.1 产品定位
一款专为安卓手机设计的本地化听记助手，用于会议场景的录音转写和纪要生成，所有数据存储在用户手机端，保护隐私安全。

### 1.2 目标用户
- 商务人士：需要记录会议内容
- 学生群体：课堂录音转写
- 记者/编辑：采访录音整理
- 任何需要语音转文字的用户

### 1.3 核心价值
- **隐私安全**：所有数据本地存储，不上传云端
- **多语言支持**：支持普通话和英语识别
- **灵活输出**：支持实时录音、文件上传、文本输入三种方式生成纪要
- **大文件支持**：单文件最大500M，满足长时间会议需求
- **灵活付费**：免费体验 + 订阅 + 买断模式，价格亲民

## 2. 功能需求

### 2.1 核心功能

#### F1: 实时录音转文字
- **描述**：用户点击录音按钮，实时录音并转写为文字
- **输入**：麦克风实时音频流
- **处理**：
  - 支持普通话、英语自动识别或手动切换
  - 实时显示转写文字
  - 可暂停/继续录音
- **输出**：
  - 原始录音文件（AMR/MP3格式）
  - 转写文字（可编辑）
  - 一键生成会议纪要
- **限制**：单文件最大500M，超限提示"已达文件大小限制，是否开始新录音？"

#### F2: 上传录音转文字
- **描述**：用户从手机选择已录制的音频文件，上传后转写为文字
- **支持格式**：MP3, WAV, AMR, AAC, M4A
- **文件大小限制**：最大500M
- **处理流程**：
  1. 文件格式验证
  2. 文件大小检查
  3. 音频转写（进度显示）
  4. 生成纪要

#### F3: 输入文本生成纪要
- **描述**：用户直接输入或粘贴文本，APP自动生成结构化会议纪要
- **输入方式**：
  - 键盘输入
  - 粘贴文本
  - 从其他APP分享导入
- **输出**：结构化纪要（包含时间、地点、参会人、议题、结论等）

#### F4: 数据管理与保存
- **描述**：用户可选择保存录音、转写文字、纪要中的任意组合
- **保存选项**：
  - [ ] 保存录音文件
  - [ ] 保存转写文字
  - [ ] 保存会议纪要
- **存储位置**：手机本地数据库（Room）+ 可选导出到文件系统
- **管理方式**：
  - 列表展示所有记录
  - 支持搜索（按关键词、日期、标签）
  - 支持删除、编辑、分享

### 2.2 语音识别需求

#### 识别引擎
- **在线识别**：使用Android SpeechRecognizer API（需联网）
- **离线识别**：集成Vosk离线语音识别SDK（可选，需下载语言模型）
- **语言支持**：
  - 普通话（简体中文）
  - 英语（美国/英国）

#### 识别精度要求
- 普通话：≥ 85%
- 英语：≥ 80%

### 2.3 UI/UX需求

#### 主界面
- 三个核心功能入口（卡片式设计）：
  1. 实时录音
  2. 上传录音
  3. 输入文本
- 最近记录快捷入口
- 设置按钮（语言切换、识别引擎选择）
- **剩余免费次数提示**

#### 录音界面
- 大号录音按钮（醒目易点）
- 实时显示：
  - 录音时长
  - 文件大小
  - 转写文字（滚动显示）
- 控制按钮：暂停/继续、停止、取消

#### 转写结果界面
- 三个标签页：
  1. 转写文字（可编辑）
  2. 会议纪要（自动生成）
  3. 原始录音（可播放）
- 底部操作栏：保存、分享、重新生成纪要

#### 历史记录界面
- 列表展示（按时间倒序）
- 每个记录显示：
  - 标题（自动生成或用户自定义）
  - 日期时间
  - 时长
  - 标签
- 支持搜索和筛选

### 2.4 付费订阅功能

#### F5: 免费体验机制
- **描述**：新用户首次使用可享受前10次免费转写，无时长限制
- **规则**：
  - 每个设备/账号限享一次免费体验
  - 免费次数使用完毕后提示升级订阅
  - 免费体验期间功能完整可用

#### F6: 订阅套餐管理
- **套餐类型**：
  | 套餐 | 价格 | 内容 |
  |-----|------|-----|
  | 轻量版 | ¥9.9/月 | 每月100分钟转写时长 |
  | 专业版 | ¥19.9/月 | 每月无限转写 + 全部高级功能 |
  | 买断版 | ¥399 | 永久使用 + 全部高级功能 |

- **功能对比**：
  | 功能 | 免费版 | 轻量版 | 专业版 | 买断版 |
  |-----|-------|--------|--------|--------|
  | 基础转写 | ✓ (10次) | ✓ | ✓ | ✓ |
  | 会议纪要生成 | ✓ | ✓ | ✓ | ✓ |
  | 高级纪要模板 | ✗ | ✗ | ✓ | ✓ |
  | PDF导出 | ✗ | ✗ | ✓ | ✓ |
  | 云备份 | ✗ | ✗ | ✓ | ✓ |
  | 使用期限 | 永久 | 订阅期 | 订阅期 | 永久 |

#### F7: 支付与订单管理
- **支付方式**：微信支付、支付宝
- **订单记录**：显示所有订阅订单和买断记录
- **自动续费**：支持开启/关闭自动续费
- **发票申请**：支持电子发票开具

#### F8: 付费状态提示
- **剩余次数/时长显示**：在主界面和录音界面显示
- **付费引导**：免费次数即将用完时提示升级
- **付费成功通知**：订阅成功后弹出确认提示

## 3. 非功能需求

### 3.1 性能要求
- APP启动时间：< 2秒
- 语音转写延迟：< 3秒（在线）、< 5秒（离线）
- 支持后台录音（Service保活）

### 3.2 存储要求
- 所有数据存储在手机本地
- 使用Room数据库管理结构化数据
- 音频文件存储在应用私有目录
- 支持自动清理缓存（用户可配置）

### 3.3 兼容性要求
- 最低支持Android 8.0（API 26）
- 目标版本Android 14（API 34）
- 支持不同屏幕尺寸（手机）

### 3.4 权限需求
- RECORD_AUDIO（录音）
- READ_EXTERNAL_STORAGE（读取文件）
- WRITE_EXTERNAL_STORAGE（保存文件）
- INTERNET（在线语音识别、支付）
